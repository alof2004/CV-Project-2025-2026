# Project Folder Guide

The Godot project lives in `mage-chronicles/`. Below is what each folder contains.

- `mage-chronicles/`: Project root (open this folder in Godot).
- `mage-chronicles/Assets/`: Game assets like models, textures, sprites, and animations.
- `mage-chronicles/Blender Files/`: Blender files (just for storage purposes)
- `mage-chronicles/Environment/`: Environment resources (lighting/sky settings, .tres).
- `mage-chronicles/Levels/`: Level scenes
- `mage-chronicles/Scenes/`: Reusable scenes
- `mage-chronicles/Scripts/`: Scripts used in the different scenes
- `mage-chronicles/Shaders/`: Godot shader files (.gdshader).
- `mage-chronicles/Textures/`: Standalone textures and texture resources.

Other files at the repo root:
- `controls.txt`: Input map controls to play the games
- `changes.txt`: Notes on project changes.
- `Presentation.pdf`: Presentation of the last class
